# Frontend Plugins

هذا المجلد مخصص للإضافات المخصصة في الفرونت-إند.

## أمثلة على الاستخدامات:

### 📊 Chart Plugins

- إضافات مخصصة لمكتبات الرسوم البيانية
- Custom tooltips, legends, animations

### 🎨 UI Plugins

- React components قابلة لإعادة الاستخدام
- Custom hooks
- Animation libraries

### 🔌 Third-party Integrations

- Google Analytics
- Payment gateways
- Map integrations

### 📝 Editor Plugins

- WYSIWYG editors
- Rich text editors
- Code editors

## Structure Example:

```
plugins/
├── charts/
│   └── customTooltip.js
├── analytics/
│   └── googleAnalytics.js
└── hooks/
    └── useCustomHook.js
```
